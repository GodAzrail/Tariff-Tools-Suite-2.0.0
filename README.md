# Tariff Tools Suite

## 🧑‍💻 Установка

### Вариант 1: Установка как расширение (Chrome / Edge)

1. Открой браузер.
2. Перейди в:

   * Chrome: `chrome://extensions/`
   * Edge: `edge://extensions/`
3. Включи **Developer mode**.
4. Нажми **Load unpacked**.
5. Выбери папку проекта.

## 📦 О проекте

**Tariff Tools Suite** — это браузерное расширение для внутренней страницы конфигурации тарифов. Оно добавляет инструменты для:

* экспорта тарифов;
* массового создания тарифов из Excel;
* массового обновления тарифов из Excel;
* массового переименования тарифов;
* кастомизации отображения карточек тарифов;
* ускоренного заполнения цен;
* улучшения и ресайза служебных диалогов.

Проект работает как расширение для страниц `tariffs` и `del-orgs` и подключает разные content scripts в зависимости от маршрута.

## 📂 Описание файлов

### Служебные и конфигурационные файлы

#### `manifest.json`

Главный manifest расширения Chrome/Edge. Описывает название, версию, разрешения, popup, и какие content scripts подключаются на страницах `tariffs` и `del-orgs`.

#### `popup.html`

Разметка всплывающего окна расширения. Показывает пользователю базовую информацию о состоянии расширения.

#### `popup.js`

Логика popup-окна. Проверяет текущую вкладку и сообщает, открыта ли поддерживаемая страница: `/configurator/tariffs` или `/configurator/del-orgs`.

#### `plugin-core.js`

Общий внутренний utility-слой проекта. Содержит:

* определение маршрутов;
* helpers для `localStorage`;
* DOM-утилиты;
* управление сайдбарами;
* вычисление responsive-высоты;
* диагностику состояния модулей.

#### `content.css`

Общие стили, которые подмешиваются в поддерживаемые страницы. Используется для оформления встроенных элементов интерфейса расширения.

#### `xlsx.full.min.js`

Минифицированная библиотека SheetJS для чтения и генерации Excel-файлов. Используется в инструментах импорта, создания, обновления и переименования тарифов.

#### `.gitattributes`

Служебный файл Git с настройками обработки файлов в репозитории.

#### `icons/icon-48.png`, `icons/icon-128.png`

Иконки браузерного расширения для панели браузера и страницы расширений.

### Инструменты для работы с тарифами

#### `tariff-tools-content.js`

Крупный основной content-script для страницы тарифов. Отвечает за экспортный интерфейс, кнопки навигации, сайдбар состояния, логи выполнения, сохранение состояния в `localStorage` и координацию процесса экспорта тарифов.

#### `tariff-page-collector.js`

Сборщик данных с отдельной страницы/модалки тарифа. Ждёт готовности модального окна, извлекает полные данные тарифа и сохраняет их в `localStorage` как `collected_tariff_data`, чтобы основной модуль экспорта мог продолжить обработку.

#### `tariff-creator.js`

Модуль массового создания тарифов из Excel. Загружает файл, показывает конфигурационный и прогресс-сайдбар, ведёт лог и пошагово создаёт новые тарифы через интерфейс сайта.

#### `tariff-updater.js`

Модуль массового обновления существующих тарифов из Excel. Считывает Excel, управляет конфигурацией импорта, логирует процесс и по шагам применяет изменения к уже существующим тарифам.

#### `tariff-renamer.js`

Модуль массового переименования тарифов через Excel. Даёт шаблон для выгрузки текущих названий, читает файл с колонками старого и нового имени и автоматически переименовывает тарифы с отображением прогресса и логов.

#### `tariff-customizer.js`

Модуль кастомизации страницы тарифов. Добавляет персональную организацию карточек тарифов: избранное, VIP/супер-категории, закрепление, пользовательский порядок, drag-and-drop, восстановление раскладки из `localStorage` и адаптивную переразметку колонок.

#### `tariff-tools-patch.js`

Небольшой устаревший патч-заглушка. Сейчас фактически ничего не меняет и только пишет в лог, что отдельный patch больше не нужен, потому что дедупликация логов перенесена в основной модуль.

### UI-утилиты и диалоги

#### `autofill-content.js`

Инструмент быстрого заполнения цен в диалоге тарифов. Ищет таблицу с полями цен, встраивает компактную панель управления, позволяет вставить значения пачкой, применить их ко всем строкам и скопировать результат в Excel-совместимом табличном виде.

#### `resize-core.js`

Базовый модуль ресайза диалогов. Даёт общие функции для расчёта высоты, позиционирования окна и наблюдения за появлением диалогов. При этом на странице `/configurator/tariffs` изменение размеров специально блокируется, а основной сценарий рассчитан на совместимость со страницей `del-orgs`.

#### `resize-dialog.js`

Скрипт для увеличения и улучшения диалога `select-volume-weight-dlg` — окна, связанного с созданием набора МГХ/сетки. Определяет нужный диалог, подбирает responsive-высоту и улучшает прокрутку и компоновку содержимого.

#### `resize-intervals-dialog.js`

Скрипт для окна `select-invervals-dlg` на странице `del-orgs`. Делает диалог интервалов доставки выше и удобнее для работы, ищет строки/контейнеры и подстраивает layout под большой объём данных.

#### `resize-branches-dialog.js`

Скрипт для окна `select-branches-dlg` на странице `del-orgs`. Отвечает за улучшение диалога выбора филиалов обслуживания: увеличивает окно, перераспределяет внутренние блоки и делает список удобнее для просмотра.

#### `resize-schedule-dialog.js`

Скрипт для окна `select-schedule-dlg`. Работает с таблицей недельного расписания, наблюдает за состоянием диалога и улучшает его размер и поведение для более удобного редактирования графика.

## 🔗 Как связаны модули между собой

1. `manifest.json` подключает нужные скрипты на нужные маршруты.
2. `popup.html` + `popup.js` дают пользователю быстрый статус по активной вкладке.
3. `tariff-tools-content.js` управляет экспортным UI и общим процессом экспорта.
4. `tariff-page-collector.js` собирает детальные данные тарифа и возвращает их через `localStorage`.
5. `tariff-creator.js`, `tariff-updater.js` и `tariff-renamer.js` обрабатывают Excel-сценарии массовых операций.
6. `tariff-customizer.js` отвечает за визуальную организацию карточек тарифов.
7. `autofill-content.js` и resize-скрипты улучшают UX отдельных окон и форм.
8. `xlsx.full.min.js` используется всеми Excel-сценариями.
